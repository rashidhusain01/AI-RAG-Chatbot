# 🤖 AI RAG Chatbot

An AI-powered Retrieval Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions based on the uploaded content.

The system extracts text from PDFs, creates embeddings, stores them in a FAISS vector database, retrieves relevant information, and generates accurate answers using an LLM.

---

## 🚀 Features

- Upload PDF documents
- Extract text from PDF files
- Split documents into smaller chunks
- Generate embeddings using AI embedding models
- Store embeddings using FAISS vector database
- Semantic search for relevant information
- AI-generated answers using GPT-4o-mini
- FastAPI backend API
- Secure API key handling using environment variables
